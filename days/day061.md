3. 타임리프 문법
    - 텍스트 출력 - th:text
    ```
        <p th:text="${msg}"></p>
        <p th:text="${nums}"></p>
    ```
    - 조건문 - th:if
        - 특정 조건
        ```
        <p th:if="${10>3}">10은 3보다 커요</p>
        <p th:if="${10<3}">10은 3보다 작아요</p>
        ```
        - $ 변수, #타임리프에서 제공하는 값 - 유틸리티 객체식
        ```
        <p th:if="${#lists.size(nums) > 3}">데이터가 3개이상</p>
        ```
    - 반복문 - th:exch
    ```        
    <ul>
        <li th:each="n : ${nums}" th:text=${n}></li>
    </ul>
    ```
        - 문자열 연결 : |
        ```
        <ul>
        	<li th:each="n : ${nums}" th:text="|숫자: ${n}"|></li>
        </ul>
        ```
    - 속성변경 - th:href, th:src, th:value
    - th:href
        - HTML 문법
        ```
        <a href="/board/list">BOARD</a>
        ```
        - 타임리프문법
            - @ = pageContext.request.contextPath 
        ```
        <a th:href="@{/board/list}">BOARD</a>
        ```
    - th:each
    ```
    <tbody>
        <tr th:each="dto : ${list}">
            <td th:text="${dto.id}"></td>
            <td th:text="${dto.btitle}"></td>
            <td th:text="${dto.appUserId}"></td>
            <td th:text="${dto.createAt}"></td>
            <td th:text="${dto.bhit}"></td>					
        </tr>
    </tbody>
    ```
