# 👤 1번 - 회원 + 권한 + 관리자(욱진)

## 1) 기능 정의

| 구분         | 사용자 기능                                                   | 관리자 기능                                                         |
| ---------- | -------------------------------------------------------- | -------------------------------------------------------------- |
| **Create** | 일반회원 가입, 제휴업체 회원가입                                       | 관리자 등록, 회원가입                                                   |
| **Read**   | 로그인, 로그아웃, 아이디 찾기, 비밀번호 찾기, 아이디 중복확인, 닉네임 중복확인, 마이페이지 조회 | 로그인, 로그아웃, 회원 조회, 회원 검색, 회원 리스트(페이징), 관리자 조회, 아이디중복검사, 닉네임중복확인 |
| **Update** | 비밀번호 변경, 프로필 수정, 관심사 수정                                  | 회원정보 수정, 회원 상태 변경, 관리자/제휴업체 회원 승인/거절, 관리자 권한 수정                |
| **Delete** | 회원탈퇴                                                     | 회원 삭제, 관리자 삭제                                                  |

### 관련테이블

* members(회원)

### 2차

※ 이메일 인증 API
※ 비밀번호 암호화
※ 자동 로그인 / 로그인 유지
※ 일일 가입자 수
※ 전체 회원 수
※ 내가신청한글보기
※ 회원 통계
※ 주소 API 연동

### 공통

| 필드명(영문)    | 필드명(한글) | TYPE     | PK | FK | 제약조건     | 기본값               | 설명        |
| ---------- | ------- | -------- | -- | -- | -------- | ----------------- | --------- |
| delete_yn  | 삭제여부    | BOOLEAN  |    |    | NOT NULL | N                 | 논리삭제      |
| created_at | 생성일시    | DATETIME |    |    | NOT NULL | CURRENT_TIMESTAMP | 생성 시간     |
| created_by | 생성자 아이디 | INT      |    | O  | NOT NULL |                   | 생성자 회원 ID |
| updated_at | 수정일시    | DATETIME |    |    | NOT NULL | CURRENT_TIMESTAMP | 수정 시간     |
| updated_by | 수정자 아이디 | INT      |    | O  | NOT NULL |                   | 수정자 회원 ID |
